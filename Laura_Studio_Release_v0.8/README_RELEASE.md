# Laura Image Studio v0.8 - Viral Video Production Edition (2026)

## Features:
- **SOTA Video Face Drive**: LivePortrait v2 (flicker-free).
- **Cinema Upscale**: SUPIR-Video + RIFE v4 (Spatial/Temporal hybrid).
- **Directed Motion**: Wan 2.2 + Motion-Control prompts.
- **Auto-VRAM Tiers**: Optimized for RTX 40-series and below.
- **Character LoRA Loader**: Single-click influencer generation.

## Installation:
1. Copy `Laura_Image_Studio` to your `ComfyUI/custom_nodes/` directory.
2. Install dependencies: `pip install -r requirements.txt` (if applicable).
3. Load `workflows/viral_video_production_v1.json` in ComfyUI.
